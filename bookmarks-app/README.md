# BookmarksApp

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 8.3.28.

## Run app
Navigate to directory "bookmarks-app"
- run `npm install`
- run `npm start`
- Navigate to `http://localhost:4200/`.
